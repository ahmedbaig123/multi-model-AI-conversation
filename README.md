# Multi-Model AI Conversation

A Python project that simulates a discussion between multiple local LLMs using Ollama. Each model has a unique personality and maintains its own conversation history over multiple rounds.

## Features
- Multi-agent AI conversation
- Three local Ollama models
- Distinct AI personalities
- Multi-round discussions
- Conversation logging (JSON)

## Models
- Llama 3.2
- DeepSeek R1
- Qwen 2.5
